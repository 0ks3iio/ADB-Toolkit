## Installation


```bash
git clone https://github.com/ASHWIN990/ADB-Toolkit.git

cd ADB-Toolkit

sudo chmod +x install.sh

sudo ./install.sh "or" sudo bash install.sh
```

```bash
sudo ./ADB-Toolkit.sh
```
